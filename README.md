UC GPU Rendering Optimization (OpenGL → Vulkan)

📌 Overview

UC GPU Rendering Optimization is a Magisk module designed to enhance rendering performance by prioritizing Vulkan over OpenGL in the Android graphics pipeline.

This module aims to improve GPU efficiency, reduce overhead, and provide smoother rendering in supported applications and games.

---

⚙️ Features

- Vulkan rendering over OpenGl
- Reduced rendering overhead
- Improved GPU efficiency
- Better frame stability in compatible apps
- Optimized graphics pipeline behavior

---

🔧 How It Works

This module applies system-level tweaks that influence Android's rendering pipeline:

- Adjusts system properties related to GPU rendering 
- Optimizes driver-level behavior (device dependent)
-  -improve frame pacing and reduce CPU-GPU bottlenecks

«⚠️ Note: Android does not allow forcing Vulkan globally for all apps. This module works by influencing system behavior, not overriding app-level APIs.»

---

📱 Compatibility

- Devices with Vulkan support (Android 10+ recommended)
- Works best on:
  - Mali GPUs
  - Adreno GPUs (partial benefit)
- Magisk v20.4+
- Ksu - Required - Meta Module 
---

📦 Installation

1. Download the module ZIP
2. Open Magisk app
3. Go to Modules → Install from storage
4. Select the ZIP file
5. Reboot your device
